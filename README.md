## Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/Hotmann/Portfolio.git
   ```

2. Navigate to the project directory:
   `cd Portfolio`
3. Install dependencies:
   `npm install`
4. Start the development server:
   `npm start`
